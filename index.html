About My PhD

Title

Interactive query lanquages for data exploration using machine learning principles

Abstract

This PhD aims at developing new approaches for interactive big data exploration by revisiting classical database approaches due to prohibitive running times, memory consumption, and volumes of the outputs. The key idea is to consider the data analyst as an essential link in the data exploration process, without making any hypothesis neither on the size of the data space nor on the size of the pattern space to be analyzed. We intend to guide and help the data analyst to both elicit zones of interest in the data space and to navigate in the pattern space, based on declarative approaches originating from database and artificial intelligence. We plan to propose high-level prototypes (e.g. from declarative languages or convenient user interfaces) to data analysts, hiding as much as possible the inherent complexity of big data analytics. An example of this is a SQL query completion prototype aiming at guiding databases exploration by suggesting possible completions for a given SQL query.
